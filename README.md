# AJ Academy

| Folder | Purpose |
|--------|---------|
| **AJ_Academy_OS** | Next.js app (deploy this folder on Vercel) |
| **AJ_Academy_SB** | Supabase SQL scripts |

## Deploy on Vercel

**Required:** Vercel → Settings → General → **Root Directory** = `AJ_Academy_OS`

Without this, deploy shows Ready but the site returns **404**.

See [AJ_Academy_OS/DEPLOYMENT.md](AJ_Academy_OS/DEPLOYMENT.md).
